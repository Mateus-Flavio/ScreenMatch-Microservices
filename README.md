# ScreenMatch Microservices

Projeto desenvolvido com Java e Spring Boot utilizando arquitetura baseada em microserviços.

## Funcionalidades implementadas

* Cadastro de usuários
* Login de usuários
* Recuperação de senha
* Cadastro de filmes
* Listagem de filmes
* Comunicação entre microserviços via API REST
* Integração com PostgreSQL

## Tecnologias utilizadas

* Java 21
* Spring Boot
* Spring Security
* Spring Data JPA
* PostgreSQL
* Thymeleaf
* Maven
* Git e GitHub

## Arquitetura

### usuario-service

Responsável pelo cadastro, autenticação e gerenciamento de usuários.

### filmes-service

Responsável pelo cadastro e gerenciamento dos filmes.
